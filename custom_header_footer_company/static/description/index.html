Expected Wkhtmltopdf Version : 0.12.1

Key features:
  • Adding Header and Footer image On All Reports.

This module adds some fields on company configuration.

Configuration for custom header footer on all reports:
Use images for all report : Boolean Check Box
Paper format : All Report Paper(paper formate)
Header Image : Upload file
Footer Image : Upload file
Above configuration Option are available inside company form view's general information tab.
Notes
*Custom header footer only applicable on those reports which have external header footer layout.
*After above configurations standard report templates layout will not be applicable.
